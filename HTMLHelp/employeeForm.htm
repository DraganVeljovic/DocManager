Search for documents

All search criteria are optional, you can select on or all of them.
Please choose which one you want by checking the checkbox on the left.
You are able to search from documents for which you have either read or write permission.

Title

Check this one if you want to search documents by title.

Type

Check this one if you want to search documents by type.

Creation interval

Check this one if you want to search documents by time of its creation. You should choose start date and end date of that interval.

Archived

Check this one if you want to search archived documents.

Search...

Press Search to search for documents by selected criteria.
