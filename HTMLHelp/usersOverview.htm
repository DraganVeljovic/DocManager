User overview and search

THIS OPTION IS ONLY AVAILABLE FOR ADMINISTRATORS

Open Users overview dialog

From menubar choose User->Users overview

All search criteria are optional, you can select on or all of them.
Please choose which one you want by checking the checkbox on the left.

Username

Check this one if you want to search users by username.

First name

Check this one if you want to search users by first name.

Last name

Check this one if you want to search users by last name.

E-mail

Check this one if you want to search users by e-mail.

Telephone

Check this one if you want to search users by telephone.

Office number

Check this one if you want to search users by office number.

Search...

Press Search to search for users by selected criteria.
